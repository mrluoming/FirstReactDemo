# FirstReactDemo


## 环境要求
```
  Nodejs 8.9.4
  React v16.5.1
```

## Features

- **最新技术栈**：使用 React开发


## 代码结构
```
- src
 |___ components    可复用的组件
 |___ containers    页面容器
 |___ styles        css
 |___ test          测试
```

## 本地启动

```bash
$ git clone https://github.com/mrluoming/FirstReactDemo.git --depth=1
$ cd FirstReactDemo
$ npm install
$ npm start         # visit http://localhost:3000
```
